roFileWalker  

A read only version of FileWalker designed for use on web hosting services that do NOT allow file managers, roFileWalker can also be used as a demonstration version of PHP Navigator, dbFileWalker, as it is completely "readonly". Although fully fuctional from a PHP code point of view, it is still missing the required functions to perform said task, making hacking it useless.
